# Contributing to tonguetoquill-usaf-memo

Thank you for your interest in contributing! This project is a [Typst](https://typst.app) template for creating AFH 33-337–compliant USAF memorandums, and contributions of all kinds are welcome.

## Getting Started

1. **Clone the repo**
   ```bash
   git clone https://github.com/nibsbin/tonguetoquill-usaf-memo.git
   cd tonguetoquill-usaf-memo
   ```

2. **Install Typst** — follow the [official instructions](https://github.com/typst/typst?tab=readme-ov-file#installation).

3. **Build the example PDFs** to verify everything works:
   ```bash
   ./build.sh
   ```
   This compiles the three template files (`usaf-template`, `ussf-template`, `starkindustries`) into `pdfs/`.

## Project Layout

| Path | Purpose |
|---|---|
| `src/` | Core library — `lib.typ` re-exports the public API (`frontmatter`, `mainmatter`, `backmatter`, `indorsement`) |
| `template/` | User-facing example templates and assets |
| `fonts/` | Open-source fonts bundled with the package |
| `pdfs/` | Compiled example outputs (generated by `build.sh`) |
| `prose/` | Design documents, implementation plans, and archived analyses |
| `typst.toml` | Package manifest (name, version, entrypoint, etc.) |

## Making Changes

1. **Create a branch** off `main` for your work.
2. **Edit source files** in `src/`. The entrypoint is `src/lib.typ`.
3. **Test your changes** by compiling one or more templates:
   ```bash
   typst compile --font-path . --root . template/usaf-template.typ pdfs/usaf-template.pdf
   ```
   Or run `./build.sh` to compile all three.
4. **Visually inspect** the generated PDFs against AFH 33-337 formatting standards.

## Submitting a Pull Request

1. Open an issue first if the change is non-trivial, so we can discuss the approach.
2. Keep PRs focused — one logical change per PR.
3. Update `CHANGELOG.md` with a brief description of your change under an `[Unreleased]` heading.
4. Make sure `./build.sh` completes without errors before submitting.

## Style Notes

- The project uses [Typst](https://typst.app/docs) (`.typ` files), not LaTeX.
- AFH 33-337 "The Tongue and Quill" is the authoritative reference for formatting decisions.
- Sentence spacing helpers are available in `.github/prompts/` for Copilot-assisted editing.
